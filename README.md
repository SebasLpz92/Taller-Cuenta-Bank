# Gestor de Cuentas Bancarias y Tarjetas de Crédito

## Descripción
Este programa permite gestionar cuentas bancarias y tarjetas de crédito, incluyendo consultas de saldo, depósitos, retiros y pagos de tarjeta.

## Funcionalidad
- **CuentaBancaria**: Administra el saldo y permite depósitos, retiros y pagos de tarjeta.
- **TarjetaCredito**: Valida números de tarjeta (algoritmo de Luhn) y gestiona pagos pendientes.
- **Interfaz de Menú**: Permite interactuar con el usuario para realizar operaciones.

## Uso
Ejecuta el programa y sigue las instrucciones en pantalla para:
1. Crear una cuenta bancaria.
2. Consultar saldos.
3. Realizar transacciones.

## Ejemplo de número de tarjeta válido
```
4532015112830366
```
